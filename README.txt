Simple module that adds Open Graph (OGP) tags on PCP pages for Facebook and
other sites where you can share links.

For more information: http://ogp.me/

Once the module is enabled, it will automatically generate the tags for
PCP pages, no further configuration is required. It supports the page
title, description and PCP image.
